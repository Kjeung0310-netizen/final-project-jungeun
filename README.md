# final-project-jungeun
# ⚾ 나만의 응원 포스터 제작기 (Sports Poster Maker)

이 프로젝트는 파이썬(Streamlit)을 활용하여 사용자가 원하는 응원 문구를 입력하면, 야구장 배경 이미지와 합성하여 나만의 응원 포스터를 제작하고 즉시 다운로드할 수 있는 웹 애플리케이션입니다.

## 🚀 주요 기능
- **실시간 응원 문구 입력:** 사용자 입력에 따라 포스터 문구가 즉시 반영됨.
- **글자 크기 조절:** '작게', '중간', '크게' 옵션을 통해 사용자가 원하는 레이아웃 구성 가능.
- **포스터 파일 저장:** '📥 포스터 파일 저장하기' 버튼을 통해 결과물을 즉시 다운로드.
- **반응형 UI:** 사이드바를 활용한 설정 메뉴와 메인 화면 분리로 사용자 편의성 제공.

## 🛠 사용 기술 및 도구
- **Framework:** Streamlit
- **Library:** Pillow (이미지 처리), Requests (이미지 로드)
- **Deployment:** GitHub, Streamlit Community Cloud
- **AI Tool:** Univ.ai Assistant (코드 구조 설계, 배포 에러 디버깅, 문서 작성 지원)

## 🛠 개선 기록 (AI 활용 및 디버깅)
1. **반응형 UI 구현:** 초기 HTML 방식의 한계를 극복하기 위해 `st.columns`를 사용하여 좌우 분할 레이아웃 도입.
2. **보안 에러 디버깅:** HTML 태그 직접 삽입 시 발생하는 `Module script failed` 에러를 파이썬 기본 함수(`st.image`, `PIL`) 위주로 변경하여 해결.
3. **사용성 개선:** 외부 이미지 링크 차단 문제를 해결하기 위해 로컬 이미지를 직접 로드하는 방식으로 안정화.
